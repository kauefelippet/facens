HTML

HTML é uma linguagem de marcação utilizada para desenvolver páginas web. A sigla HTML significa HyperText Markup Language, ou seja, Linguagem de Marcação de Hipertexto. A linguagem HTML é composta por tags que definem o conteúdo e a estrutura de uma página web.

CSS

CSS é uma linguagem de estilo utilizada para definir a apresentação de páginas web. A sigla CSS significa Cascading Style Sheets, ou seja, Folhas de Estilo em Cascata. A linguagem CSS é composta por regras que definem a aparência de elementos HTML.

JavaScript

JavaScript é uma linguagem de programação utilizada para desenvolver páginas web interativas. A linguagem JavaScript é composta por instruções que são executadas no navegador do usuário. Com JavaScript, é possível criar efeitos visuais, validar formulários e manipular o conteúdo de uma página web.

PHP

PHP é uma linguagem de programação utilizada para desenvolver aplicações web dinâmicas. A sigla PHP significa PHP: Hypertext Preprocessor, ou seja, PHP: Pré-processador de Hipertexto. A linguagem PHP é composta por scripts que são executados no servidor web.

Python

Python é uma linguagem de programação utilizada para desenvolver aplicações web, científicas e educacionais. A linguagem Python é conhecida por sua simplicidade e legibilidade. Com Python, é possível criar scripts, automatizar tarefas e desenvolver sistemas complexos.
